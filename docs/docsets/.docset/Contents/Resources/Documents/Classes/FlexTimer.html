FlexTimer

@IBDesignable
class FlexTimer : CustomStringConvertible

A custom rolled timer, based on Timer ( aka NSTimer ) but with finer grained controls over starting/stopping and with incremental callout points and completion.

As with the original Timer class, once cancelled a timer will be considered invalid and cannot be reused.
